Especificaciones del Componente

El componente EmptySectionComponent es un componente diseñado para mejorar la experiencia del usuario en escenarios donde no hay datos disponibles para mostrar.

Propiedades
  • message(obligatoria): string = Mensaje que describe el estado vacío. Debe ser claro y directo para el usuario final.
  • styleClass: string = Clases CSS adicionales para personalización específica.
Uso básico:
<app-empty-section message="Hola mundo"></app-empty-section>
